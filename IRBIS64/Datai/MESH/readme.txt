Последовательность действий при создании новой версии MeSH

1. Взять из предыдущей версии первые две записи (Справочная запись и Верхний уровень)
2. В АРМе Администратор импортировать ISO-файл с новой версией MeSH c ТВП-перевформатирования mesh_marc,
   БЕЗ ФЛК И АВТОВВОДА
3. В АРМе Каталогизатор выполнить глобальную на всю БД MeSH - autoin.gbl с отключением АВТОВВОДА и АКТУАЛИЗАЦИИ
4. В АРМе Администратор сделать СОЗДАТЬ СЛОВАРЬ ЗАНОВО и потом РЕОРГАНИЗОВАТЬ ФАЙЛ ДОКУМЕНТОВ
